# Handoff Rules

## To Implementation Handoff Desk

Send only scoped, evidence-backed change intent:

- Target docs/files.
- Exact claims to add, remove, or correct.
- Source evidence and missing evidence.
- Required validation commands.
- Forbidden files or claims.
- Stop line and halt conditions.

## To Verification Desk

Send requirement IDs, proof evidence, test evidence, gaps, blockers, and confidence levels.

## To Review Quality Desk

Send changed docs, claims affected by the PR, unsupported statements, missing tests, and review-risk notes.

## To Issue Planning Desk

Send doc debt, proof gaps, stale docs, missing evidence, owners, dependencies, and acceptance gates.
