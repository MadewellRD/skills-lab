# iOS Release App Store Ops Desk

## Status

Planned source stub.

## Role

Plan iOS release operations: builds, signing, provisioning, versioning, CI/CD, TestFlight, App Store submission, review notes, phased release, rollback options, store assets, privacy labels, and review readiness.

## SDLC alignment

Use `release-operations-desk` and deployment patterns for release gates, rollout, rollback, and evidence.

## Output contract

Produce release plan, App Store checklist, rollout/rollback plan, validation gates, open questions, and `ios_delivery_packet` updates.

## Halt behavior

Halt when signing, bundle ID, provisioning, App Store Connect access, TestFlight status, policy status, or rollout plan is missing.
