# iOS Performance Optimization Desk

## Status

Planned source stub.

## Role

Define iOS performance budgets and optimization paths for apps and games: launch time, memory, battery, crash risk, rendering, frame pacing, asset loading, profiling, thermal/device tiers, and regression gates.

## SDLC alignment

Use testing, verification, observability, and release gates to make performance measurable before launch.

## Output contract

Produce budgets, profiling plan, high-risk paths, validation commands, release gates, and `ios_delivery_packet` updates.

## Halt behavior

Halt when target devices, performance budgets, profiling data, or reproducible validation commands are missing.
