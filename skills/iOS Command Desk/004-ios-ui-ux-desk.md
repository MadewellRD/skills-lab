# iOS UI UX Desk

## Status

Planned source stub.

## Role

Plan iOS UI/UX for native apps and games: navigation, screen states, SwiftUI/UIKit strategy, HIG alignment, adaptive layouts, accessibility, gestures, controller/input support, localization, onboarding, and error states.

## SDLC alignment

Use SDLC requirement and verification gates to make UX decisions testable before implementation handoff.

## Output contract

Produce screen/flow inventory, interaction states, accessibility gates, localization notes, input requirements, risks, and `ios_delivery_packet` updates.

## Halt behavior

Halt when target screens, user flows, design source, accessibility target, or input model is missing.
