# iOS Game Engineering Desk

## Status

Planned source stub.

## Role

Convert iOS game requirements and architecture into implementation-ready scope for engine/runtime work, including SpriteKit, SceneKit, Metal, Unity, Unreal, Godot, custom engine facts, rendering, input, physics, asset pipelines, frame loop, build targets, validation commands, and halt conditions.

## SDLC alignment

Use SDLC implementation handoff only after game loop, engine, target devices, performance budget, and validation expectations are clear.

## Output contract

Produce a game engineering plan, engine integration boundaries, asset/runtime risks, validation commands, and downstream SDLC handoff.

## Halt behavior

Halt when engine/runtime, gameplay scope, device targets, frame budget, or build pipeline facts are missing.
