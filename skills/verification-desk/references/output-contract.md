# Output Contract

Use downloadable Markdown for all substantial artifacts.

## File wrapper

```markdown
# [artifact title]

## How to use this file
Use this report to decide whether the scoped work is verified, blocked, or needs follow-up. Preserve the source facts, evidence map, gaps, and downstream handoff notes.

## Report
[artifact content]
```

## Artifact names

- `verification-report.md`
- `requirements-traceability-matrix.md`
- `acceptance-gate-review.md`
- `release-blocker-report.md`
- `evidence-map.md`
- `verification-handoff.md`

## Required sections

Every report must contain source facts, scope, evidence, status, gaps, and next action.
